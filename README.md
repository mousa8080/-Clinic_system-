# Clinic Management System (Backend)

A Laravel-based backend API for managing clinic operations.

## Features
- Appointment scheduling APIs
- Doctor and specialty management
- Role-based access with Laravel Sanctum
- RESTful endpoints for mobile/frontend integration

## Tech Stack
PHP, Laravel, MySQL, Sanctum, Eloquent ORM

## How to run
1. Clone repo
2. composer install
3. cp .env.example .env
4. php artisan key:generate
5. php artisan migrate
6. php artisan serve
